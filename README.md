## 📖 Introduction

**Z Note** is a lightweight note-taking application built with **Flutter**.

The app follows the **Material Design 3 (MD3)** design system and provides a clean and modern user experience.

目前支持基础的笔记管理功能：

- 创建笔记
- 查看笔记
- 编辑笔记
- 删除笔记

适合作为 Flutter 学习项目，也可以作为简单的日常记事工具。

## 🚀 Getting Started

### Requirements

- Flutter SDK
- Dart SDK

### Clone

```bash
git clone https://github.com/Mickeyzz6814/z_note.git
```

### Install dependencies

```bash
flutter pub get
```

### Run

```bash
flutter run
```
